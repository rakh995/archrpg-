# ArchRPG — Кэш жады архитектурасының симуляторы

## Сипаттамасы

ArchRPG — процессордың кэш жадысының жұмысын LRU алгоритмі арқылы модельдейтін интерактивті веб-қосымша.  
Жоба архитектуралық параметрлердің жүйенің өнімділігіне әсерін ойын элементтері арқылы визуалды түрде түсінуге көмектеседі.

## Технологиялар

- Next.js (React негізіндегі фреймворк)  
- Tailwind CSS (стильдерді жасау)  
- JavaScript (симуляция логикасы)
